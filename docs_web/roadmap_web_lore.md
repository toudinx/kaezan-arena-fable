# Roadmap Web — Lore / Codex  🟢 Claude-only

> **Etiqueta: 🟢 Claude-only.** Expande o lore do universo Kaezan em texto — regiões, facções,
> backstories, eventos. Sem skill (template inline). Leia [CLAUDE_WEB.md](CLAUDE_WEB.md) antes.
>
> **Como disparar:** *"rode o prompt LO-NN do `docs_web/roadmap_web_lore.md`"*.
> **Lê:** README.md + roster_digest.md + o doc de lore nomeado. **Escreve:** `docs_web/lore/`.

## Princípios de lore (todo prompt respeita)
- **Fiel ao tom existente:** dark-fantasy premium com humor seco nas Kaelis (ver `roster_digest.md`).
  Reaproveitar a filosofia do `docs/DESIGN_NOTES.md` (ideias > arquivos; nada de copiar IP).
- **Consistência elemental:** o universo é organizado pela matriz Holy/Physical/Death/Fire/
  Energy/Ice/Earth (uma Kaeli por elemento hoje).
- **Texto puro player-facing:** lore que pode virar tooltip, fragmento de afinidade, descrição de
  bioma — não documento de engine.
- **Não contradizer** as backstories já escritas das 7 Kaelis (estão no `Waifus.cs`; o resumo no digest).

## Itens

### [ ] LO-01 — Mapa do mundo: regiões por elemento
- **Escreve:** `docs_web/lore/regioes.md`
- **Objetivo:** definir 1 região-mãe por elemento (a "terra natal" visual já insinuada nos cenários
  ancorados do digest), com nome, clima, povo e gancho de conflito. Base p/ biomas e cenários de arte.

### [ ] LO-02 — Facções e ordens
- **Escreve:** `docs_web/lore/faccoes.md`
- **Objetivo:** detalhar as ordens já citadas na lore (Irmandade do Abismo, a escola de esgrima da
  Seren, os guardiões do planalto da Gaia, etc.) — propósito, hierarquia, relação entre elas.

### [ ] LO-03 — Cronologia: a "Longa Noite" e a alvorada
- **Escreve:** `docs_web/lore/cronologia.md`
- **Objetivo:** montar a linha do tempo que conecta os fragmentos (a convocação da Eloa "depois da
  Longa Noite", o lago negro da Velvet, etc.) num eixo coerente.

### [ ] LO-04 — Fragmentos de afinidade de uma Kaeli (recorrente)
- **Escreve:** `docs_web/lore/afinidade-<kaeli>.md`
- **Objetivo:** propor fragmentos adicionais de lore destraváveis por afinidade para uma Kaeli,
  no mesmo tom dos 4 que ela já tem. (Vira candidato a roadmap desktop p/ entrar em `Waifus.cs`.)

### [ ] LO-05 — Bestiário narrativo (recorrente)
- **Escreve:** `docs_web/lore/bestiario-<tema>.md`
- **Objetivo:** flavor text e gancho de lore para um grupo de monstros/bioma (o README cita
  bestiário e 50 monstros autorais por tier).
